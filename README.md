# 📄 Currículo Web Dinâmico

Projeto desenvolvido para a disciplina de **Desenvolvimento de Sistemas Web** no curso de **Análise e Desenvolvimento de Sistemas**.

O objetivo do projeto é criar uma página web que apresente um **currículo dinâmico**, onde os dados são carregados a partir de um arquivo **JSON** utilizando **Fetch API** e **JavaScript assíncrono (async/await)**.

---

# 🚀 Tecnologias Utilizadas

* **HTML5** → Estrutura da página
* **CSS3** → Estilização e layout
* **JavaScript (ES6+)** → Lógica da aplicação
* **Fetch API** → Requisição de dados
* **JSON** → Armazenamento das informações

---

```
Curriculo-Web

```

| **style.css**   | Estilização da interface                   |

---

# ⚙️ Funcionamento do Projeto

A aplicação funciona da seguinte forma:

1. O usuário acessa a página.
2. Ao selecionar uma pessoa, o JavaScript faz uma requisição usando **Fetch API**.
3. O arquivo **JSON** é carregado.
4. Os dados são convertidos para objeto JavaScript.
5. As informações são exibidas dinamicamente na página.

Fluxo da aplicação:

```
Usuário
   ↓
Clique no botão
   ↓
Fetch API
   ↓
Arquivo JSON
   ↓
JavaScript processa dados
   ↓
HTML atualizado dinamicamente
```
